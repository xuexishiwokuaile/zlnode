/Home/hostpitals

列出所有当前系统中保存的医疗机构和对应的信息

/Home/hostpitals/iList get

根据相应的信息（地理位置）获取对应的医疗机构列表

get

get方式

get /Home/hostpitals/iList

get方式

  • Request
  • Response

Query Parameters

  • currentPage: required(number)

    当前页码

  • pageSize: required(number)

    每页多少条数据

  • start: required(number)

    开启对应的页码

  • keyWord: (string)

    相应的搜索对应的关键词(搜索位置：名称，省，市，备注),不输入搜索到所有值

  • longitude: (number)

    经度信息，如果未发入相应的信息，哪么就显示所有信息，否则就显示一定范围内的医疗机构

  • latitude: (number)

    纬度信息，如果未录入信息，哪么就显示所有信息，否则就显示一定范围内的医疗机构
