/Home/hostpitals

列出所有当前系统中保存的医疗机构和对应的信息

/Home/hostpitals/iList.template get

根据相应的信息（地理位置）获取对应的医疗机构列表

get

get方式获取当前位置附近的医院，如果传递了相应的经度和纬度，哪么就计算相应的距离，不传递信息就显示所有的信息。例如查询点附近50公里的数据：http://sdev.whu.edu.cn/zf_wave/Home/hostpitals/iList.template?currentPage=0&pageSize=2&start=0&longitude=114.266164&latitude=30.583931

get /Home/hostpitals/iList.template

get方式获取当前位置附近的医院，如果传递了相应的经度和纬度，哪么就计算相应的距离，不传递信息就显示所有的信息。例如查询点附近50公里的数据：http://sdev.whu.edu.cn/zf_wave/Home/hostpitals/iList.template?currentPage=0&pageSize=2&start=0&longitude=114.266164&latitude=30.583931

  • Request
  • Response

Query Parameters

  • currentPage: required(number)

    当前页码

  • pageSize: required(number)

    每页多少条数据

  • start: required(number)

    开启对应的页码

  • keyWord: (string)

    相应的搜索对应的关键词(搜索位置：名称，省，市，备注),不输入搜索到所有值

  • longitude: (number)

    经度信息，如果未录入相应的信息，哪么就显示所有信息，否则就显示一定范围内的医疗机构(参考：114.266164)

  • latitude: (number)

    纬度信息，如果未录入信息，哪么就显示所有信息，否则就显示一定范围内的医疗机构(参考：30.583931)

/Home/medicalRecord

所有关于病案信息的操作的类，主要是集中在病案相关内容上

/Home/medicalRecord/iGetMedicalRecords.template get

病历列表,相应的针对不同角色看到不同的内容。如果当前人员是医院用户的话，看到的本机构的数据。当前人员是普通用户只能看到自己的数据。当前人员是管理员看到所有数据。

get

get方式获取当前数据http://sdev.whu.edu.cn/zf_wave/Home/medicalRecord/iGetMedicalRecords.template?hospital=%E5%8D%8F%E5%92%8C%E5%8C%BB%E9%99%A2&page=0&number_per_page=5&isChecked=false&classification=2&clientid=2018290899570001083&keyword=2018

get /Home/medicalRecord/iGetMedicalRecords.template

get方式获取当前数据http://sdev.whu.edu.cn/zf_wave/Home/medicalRecord/iGetMedicalRecords.template?hospital=%E5%8D%8F%E5%92%8C%E5%8C%BB%E9%99%A2&page=0&number_per_page=5&isChecked=false&classification=2&clientid=2018290899570001083&keyword=2018

  • Request
  • Response

Query Parameters

  • hospital: required(string)

    当前医院或医疗机构的中文名称

  • page: required(number)

    页码，决定获取数据起始位置

  • number_per_page: required(number)

    每页数据个数

  • isChecked: required(boolean)

    是否做了心电检查,true已经检查，false还未检查

  • classification: required(number)

    相应的角色信息，是什么角色

  • clientid: required(string)

    当前登录人员对应的id信息

  • keyword: required(string)

    相应的搜索信息，支持模糊搜索

get

get保存相应的信息：http://sdev.whu.edu.cn/zf_wave/Home/medicalRecord/iSetWave.template?waveFormId=2019027453920000001&waveDataId=2019027453920001001&id=2018290899700001088&waveViewId=2019027453920001001&waveFilterId=2019027453920000001

get /Home/medicalRecord/iSetWave.template

get保存相应的信息：http://sdev.whu.edu.cn/zf_wave/Home/medicalRecord/iSetWave.template?waveFormId=2019027453920000001&waveDataId=2019027453920001001&id=2018290899700001088&waveViewId=2019027453920001001&waveFilterId=2019027453920000001

  • Request
  • Response

Query Parameters

  • id: required(string)

    对应病历的ID

  • waveFormId: required(string)

    心脏波形图的数据ID

  • waveFilterId: required(string)

    相应的过滤算法过滤后的数据

  • waveDataId: required(string)

    心脏波形图分析之后的数据ID

  • waveViewId: required(string)

    对应的心脏波型数据
